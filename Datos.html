Informacion del proyecto

En el presente proyecto se buscara realizar un sistema de regadio con la ayuda de ARDUINO a traves de diferentes sensores creados e implementados por nosotros, con el fin de establecer un monitoreo con los valores que nos de ARDUINO.

Para esto, se creara un artefacto cuya finalidad es realizar un regadio para ciertos tipos de plantas como fase inicial para luego poder concretar el prouecto con un sistema de regadio a jardines.
